# matura-journal
Dieses Journal dokumentiert meinen Fortschritt in meiner Maturaarbeit ([Maturaarbeit-Repo](https://github.com/FriedUnique/matura-project)). </br>
Im Journal werden Tage oder Zeitabschnitte in Untertiteln angegeben


## 27.05.2023
Ich begann das Projekt in dem ich das "Market Sentiment" aus News-Artikel von Reuters herauslesen lies von meinem [Modul](https://github.com/FriedUnique/matura-project/tree/8caa3a6fa8147b459be98b2fbe9ed0cef28fbf66/NSA).
